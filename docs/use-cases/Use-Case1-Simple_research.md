# Use case 1: Simple research

## Assumptions
- The user role is a product manager
- The user has a new feature idea that she wants to research on
- The user is using Gemini to do some of the research

## Job to be done
- The user needs to document the process of research and then create a summary of the research
- To finish the research, the user needs to create a PRD for this feature 

## Expected outcome
- The user has a PRD for the feature

## Expected workflow using AI Researcher
1. The user creates a new project folder
2. The user creates a new file
3. The uer describes the general feature idea in the file
4. The user uses the AI chat window using Gemini to do some of the research
5. AI researcher documents the research in the file
6. AI researcher creates a summary of the research
7. AI researcher creates a PRD for the feature as a Markdown document in the project folder
